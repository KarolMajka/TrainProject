# Train Project
iOS app that use Train API to show list of stations

## Getting Started
If you don't have [CocoaPods](https://cocoapods.org/):
 
`$ [sudo] gem install cocoapods`
 
Then in project file:
 
`$ pod install`

## Pods used
- Alamofire
- Alamofire Object Mapper
- Cache
